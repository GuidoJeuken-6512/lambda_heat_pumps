# Historische Daten löschen

Falls Sie die historischen Daten der Lambda Wärmepumpen Integration aus Home Assistant entfernen möchten, können Sie dies über die Home Assistant Benutzeroberfläche durchführen.


###     Entwicklerwerkzeuge öffnen

<div style="display: flex; gap: 20px; align-items: flex-start; margin: 20px 0; flex-wrap: wrap;">
  <div style="flex: 0 0 50%; min-width: 300px;">
    <img src="../../assets/historische_daten_loeschen_de.png" alt="Integration löschen" style="width: 100%; height: auto; border-radius: 8px;">
    <img src="../../assets/historische_daten_loeschen2_de.png" alt="Ausgewählte Statistiken löschen" style="width: 100%; height: auto; border-radius: 8px; margin: 20px 0;">
  </div>
  <div style="flex: 1; min-width: 300px;">
    <ol>
      <li><strong>Öffnen Sie Home Assistant:</strong>
        <ul>
          <li>Gehen Sie zu <strong>Entwicklerwerkzeuge</strong> → <strong>Statistik</strong></li>
          <li>Suchen oder filtern Sie nach Entitäten, die Sie löschen wollen</li>
          <li>Selektieren Sie alle Etitäten, die gelöscht werden sollen</li>
          <li>Den Button "ausgewählte Statistiken löschen" nutzen um alle historischen Daten der selektierten Entitäten zu löschen</li>
          
        </ul>
      </li>
    </ol>
  </div> 
</div>

## Integration erneut hinzufügen

Falls Sie die Integration später erneut hinzufügen möchten, können Sie dies über die [Initiale Konfiguration](initiale-konfiguration.md) durchführen. Alle Einstellungen müssen neu konfiguriert werden.
